# `$registry` ontology root

Catalog namespace for the **exoas-registry** AssetSpace (EKA 16-repo audience-layered, RFC 01a83de8 SDK-model). Holds `exo__AssetSpace` descriptors (git-url `exo__AssetSpace_source`, `exo__AssetSpace_dependsOn` DAG per D18) enabling `bootstrap`/`assetspace-add`/`profile-apply` clone-on-demand. Rebuilt for EKA P7 (supersedes exoas-kitelev-registry which described the legacy schema).
